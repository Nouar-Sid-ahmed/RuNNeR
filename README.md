# Summer Project de Nouar_a

## installation :

### Rapide pour `Linux`:
```bash
sudo bash summer-project/installe.sh
```

### Cibler :

```bash
sudo apt-get install libsdl2-dev libsdl2-image-dev -y
```
puit :
```bash 
cd summer-project
```
Une foit dans le dossier du jeu 2 choix s'offre à vous :
 * installation `Linux`
```bash
sudo make
sudo make clean
```
 * installation classic
 ```bash
 sudo make mac
 ```
